# Debt_Tracker
Debt Tracker (React + Tailwind)
A lightweight, offline-first mobile web application to track debts and credits (INR). Built as a single HTML file using React and Tailwind CSS, designed to be hosted freely on GitHub Pages.

Account Management: Add up to 50 contacts/accounts.

Dual Tracking: Track both "I Owe" and "They Owe Me" amounts.

Payment History: Record partial or full payments with date stamps.

Visual Dashboard:

Monthly payment charts.

Cumulative debt trends.

Net Balance summary.

Privacy First: All data is stored in your browser's Local Storage. No data is sent to any server.

Data Portability: Export all transaction history to CSV.

Mobile Optimized: App-like feel with bottom navigation and touch-friendly controls.

ðŸ”’ Privacy & Data Persistence
This application uses localStorage to save your data.

Where is data stored? On your specific device and browser.

Is it synced? No. If you switch phones, the data will not follow you.

Backup: Use the "Export Data (CSV)" button regularly to backup your records.

Warning: Clearing your browser cache/data will delete your records.
